# Browser_GPT_agent
An AI agent that performs actions in the browser.
реализованны суб агенты 
яндекс почта
умеет удалять читать и анализировать письма
умеет писать и отправлять кому либо письма
умеет отвечать на письма
яндекс лавка
ищет и кладет товары в корзину
аналирует товары перед ответом и может дополнять корзину
hh ru
умеет анализировать профиль и резюме 
пользуется поиском и критериями 
читает вакансии и откликается 

из минусов: не успел потестировать агента на hhru не хватило времени.
минимальная эврика агенту дана но есть всегда шанс не хватки шагов либо застрять в вертске

перед запуском нужно настроить файл .env
OPENAI_API_KEY= #указываете ключ агента
PROXY= #указываете прокси для работы  с агентом
BROWSER_PROXY= #и указываете применения прокси если false то прокси будет применяться только у агента (для ру сайтов нужно false)



иструкция по запуску:
я сделал батники по этому 
1 install.bat для установки зависимостей и библиотек
2 start.bat создается вирт окружение и папка с браузером и запускается сам скрипт
3 для удобства сделал Chrome.bat он не обязателен но если нужно открыть браузер без агента то можно через батник
